# Ringle Voice Insights (RVI)

링글 AI 서비스 관련 앱스토어 리뷰, 블로그, 커뮤니티 VOC를 실시간 수집·분석하여 대시보드로 제공하는 MVP 프로젝트입니다.

## 구성

- `index.html`: 메인 대시보드
- `pages/`: 데이터 소스, VOC 탐색, 리포트
- `styles/`: CSS 변수/공통/대시보드/컴포넌트 스타일
- `scripts/`: 설정, 데이터 로딩, 차트, 필터, 유틸
- `data/mock-data.json`: 개발용 목업 데이터

## 실행 방법

로컬에서 정적 서버로 실행하거나 `index.html`을 직접 열어 확인합니다.

## 기능

- 반응형 레이아웃 (모바일/태블릿/데스크톱)
- 다크/라이트 모드 토글
- Chart.js 기반 트렌드/감성 차트
- VOC 탐색 필터링

## 배포

GitHub Pages에 업로드하여 정적 호스팅 가능합니다.
